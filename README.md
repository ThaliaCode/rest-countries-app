# REST Countries App 🌍

Aplicación web que muestra información de países del mundo, permitiendo buscar por nombre, filtrar por continente y cambiar entre modo claro y modo oscuro.

## Tecnologías usadas

- HTML
- Tailwind CSS
- JavaScript
- Git
- GitHub

## Funcionalidades

- Visualización de países en tarjetas
- Búsqueda de países por texto
- Filtro por continente
- Dark Mode
- Diseño responsive
- Manipulación del DOM con JavaScript

## Lo que aprendí

En este proyecto practiqué la creación de interfaces responsivas, el uso de eventos en JavaScript, la manipulación del DOM, filtros dinámicos y organización de código.
